## 4. 功能特点

### 搜索功能
- 输入 Tracklist ID（完整或部分）
- 输入上传者用户名
- 输入上传者昵称
- 输入 Tracklist 标题
- 实时过滤结果

### 视觉反馈
- 当前选中的 tracklist 会有蓝色边框高亮
- 鼠标悬停时显示紫色边框
- 默认 tracklist 带有"官方"标签
- 当前使用的 tracklist 带有"当前"标签

### 用户信息
- 显示上传者头像（如果有）
- 显示上传者昵称或用户名
- 显示歌曲数量
- 显示上传时间
- 显示 Tracklist ID（前 8 位）

### 响应式设计
- 适配桌面端和移动端
- 弹窗最大高度 85vh，内容可滚动
- 卡片式布局，清晰易读

## 5. 使用流程

1. **查看可用的 Tracklist**
   - 点击歌单区域的选择按钮
   - 浏览所有可用的 tracklist

2. **搜索特定 Tracklist**
   - 在搜索框输入关键词
   - 实时过滤结果

3. **切换 Tracklist**
   - 点击任意 tracklist 卡片
   - 歌曲列表自动更新
   - 播放器状态保持不变

4. **上传自己的 Tracklist**
   - 点击"上传我的 Tracklist"按钮
   - 粘贴歌单文本
   - 点击"解析歌单"
   - 确认后上传

5. **分享 Tracklist**
   - 每个 tracklist 有独立的 ID
   - 可以通过 ID 快速搜索
   - 方便分享给其他用户
