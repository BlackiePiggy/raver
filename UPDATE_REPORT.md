## 🚀 如何体验新功能

### 1. 启动项目

确保服务正在运行：

```bash
# 终端1 - 后端
cd server
pnpm dev

# 终端2 - 前端
cd web
pnpm dev
```

### 2. 访问主页

打开浏览器访问: **http://localhost:3002**

你会看到：
- 全新设计的主页
- 实时数据统计
- 即将开始的活动
- 热门DJ展示
- 功能介绍卡片

### 3. 测试打卡功能

#### 活动打卡：
1. 点击任意活动卡片进入详情页
2. 点击右侧的"打卡签到"按钮
3. 看到成功提示后，选择是否查看打卡记录

#### DJ打卡：
1. 点击任意DJ卡片进入详情页
2. 点击右侧的"打卡"按钮
3. 看到成功提示后，选择是否查看打卡记录

#### 查看打卡记录：
1. 点击主页的"打卡集邮"卡片
2. 或访问 http://localhost:3002/checkins
3. 查看所有打卡记录
4. 可以按类型筛选（全部/活动/DJ）
5. 可以删除打卡记录
