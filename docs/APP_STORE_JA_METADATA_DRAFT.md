## Description

RaveHub は、電子音楽ファンのためのイベント発見、DJ / Set ディスカバリー、チェックイン、コミュニティ投稿アプリです。

主な機能：

- イベント、DJ、Set、ニュース、学習コンテンツを検索・閲覧
- イベントへのチェックイン、写真、感想、評価の記録
- フィード投稿、コメント、メッセージ、Squad での交流
- 気になる DJ や Brand のフォローと通知
- 不適切なコンテンツやユーザーの通報、ブロック
- アカウント状態、異議申し立て、アカウント削除をアプリ内で管理

安全性とプライバシー：

通報、ブロック、アカウント制限、異議申し立て、アカウント削除の導線をアプリ内に用意しています。アカウント削除後はログイントークンとプッシュトークンを失効し、プロフィール情報を匿名化します。公開コンテンツ、監査、法令対応、安全管理に必要な記録は、プライバシーポリシーに従って最小限の範囲で保持される場合があります。

日本語のプライバシーポリシー、利用規約、コミュニティガイドライン、データ関連の請求、問い合わせページを提供しています。

## Review Notes Summary

アプリ内の主な審査導線：

- アカウント削除：`プロフィール -> 設定 -> アカウント安全 -> アカウントを削除`
- 通報：投稿、ユーザープロフィール、チャット設定、イベント、DJ、Set、Learn コンテンツのメニューから通報
- ブロック：ユーザープロフィール、チャット設定、通報完了フロー、プライバシー設定のブロックリスト
- アカウント状態と異議申し立て：`プロフィール -> 設定 -> アカウント安全`
- 通知設定：`プロフィール -> 設定 -> 通知`
- 位置情報：イベント検索、チェックイン、Squad 位置共有など、ユーザー操作に応じて利用

審査用テストアカウントと管理画面 URL は、提出時に App Store Connect の Review Notes に追記してください。
