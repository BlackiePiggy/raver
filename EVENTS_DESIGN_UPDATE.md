## 🎨 视觉设计细节

### 颜色使用

**背景**
```css
bg-primary: #000000      /* 页面背景 */
bg-elevated: #1d1d1f     /* 卡片背景 */
bg-glass: rgba(29, 29, 31, 0.72)  /* 毛玻璃 */
```

**文字**
```css
text-primary: #f5f5f7    /* 标题 */
text-secondary: #a1a1a6  /* 正文 */
text-tertiary: #6e6e73   /* 辅助 */
```

**强调色**
```css
primary-blue: #3B82F6    /* 主按钮 */
accent-green: #10B981    /* 状态标签 */
```

### 圆角系统

```css
rounded-3xl: 24px        /* 大卡片 */
rounded-2xl: 16px        /* 中卡片 */
rounded-xl: 12px         /* 小卡片 */
rounded-full: 9999px     /* 按钮、标签 */
```

### 间距系统

```css
section padding: py-16 px-6
card padding: p-8
gap: gap-12 (主要区域)
gap: gap-6 (次要区域)
```

### 阴影系统

```css
shadow-apple-lg: 0 8px 32px rgba(0, 0, 0, 0.4)
border: border-border-secondary
hover: border-border-primary
```

## ✨ 交互设计

### 悬停效果

**卡片**
- 边框颜色变化
- 阴影增强
- 图片缩放（scale-110）
- 标题颜色变化

**按钮**
- 背景颜色过渡
- 阴影增强
- 无缩放（保持稳定）

**链接**
- 颜色过渡
- 图标颜色变化
- 背景色变化

### 动画效果

**页面加载**
```css
animate-fade-in: 淡入
animate-slide-up: 上滑
animation-delay: 错开动画
```

**图片**
```css
transition: transform 700ms
hover: scale-110
```

**过渡**
```css
duration: 300-500ms
timing: ease-out
```

## 📱 响应式设计

### 断点

```css
Mobile: < 768px
Tablet: 768px - 1024px
Desktop: > 1024px
```

### 布局调整

**列表页**
- Desktop: 横向卡片（图片+内容）
- Mobile: 纵向卡片（图片在上）

**详情页**
- Desktop: 主内容 + 侧边栏
- Mobile: 单栏布局
- CTA按钮: Desktop右下角，Mobile底部

### 字体缩放

```css
Desktop: text-7xl (72px)
Mobile: text-5xl (48px)
```
