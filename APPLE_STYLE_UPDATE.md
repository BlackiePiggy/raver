# 🍎 Apple Style Design Update

更新时间: 2026-03-16

---

## ✨ 设计理念

将Raver平台改造为**苹果官网风格的暗黑高级版本**，采用：

- **极简主义** - 大量留白，聚焦内容
- **大字体** - 清晰的视觉层次
- **流畅动画** - 优雅的过渡效果
- **毛玻璃效果** - 现代感的背景模糊
- **圆角设计** - 柔和的视觉体验
- **精致阴影** - 立体感的层次

---

## 🎨 设计更新

### 1. 颜色系统

**背景色（纯黑系）**
```css
bg-primary: #000000      /* 纯黑背景 */
bg-secondary: #0a0a0a    /* 次级背景 */
bg-tertiary: #161616     /* 第三级背景 */
bg-elevated: #1d1d1f     /* 卡片背景 */
bg-glass: rgba(29, 29, 31, 0.72)  /* 毛玻璃效果 */
```

**文字色（苹果标准）**
```css
text-primary: #f5f5f7    /* 主要文字 */
text-secondary: #a1a1a6  /* 次要文字 */
text-tertiary: #6e6e73   /* 辅助文字 */
```

**边框色**
```css
border-primary: #424245   /* 主要边框 */
border-secondary: #2c2c2e /* 次要边框 */
```

### 2. 字体系统

**SF Pro Display 风格**
```css
font-family: -apple-system, BlinkMacSystemFont, 'SF Pro Display', 'Inter'
```

**字体大小**
- Display: 80px (大标题)
- Display-sm: 56px (中标题)
- 标准层次: 5xl, 3xl, 2xl, xl, base

### 3. 圆角系统

**统一使用更大的圆角**
- 按钮: `rounded-full` (完全圆角)
- 卡片: `rounded-2xl` (16px)
- 输入框: `rounded-xl` (12px)

### 4. 阴影系统

**苹果风格阴影**
```css
shadow-apple: 0 4px 16px rgba(0, 0, 0, 0.3)
shadow-apple-lg: 0 8px 32px rgba(0, 0, 0, 0.4)
```

### 5. 动画系统

**三种核心动画**
- `animate-fade-in` - 淡入效果
- `animate-slide-up` - 上滑效果
- `animate-scale-in` - 缩放效果

**动画时长**
- 标准: 300ms
- 长动画: 500-700ms
- 缓动函数: cubic-bezier(0.4, 0, 0.2, 1)

## 🎯 核心特性

### 1. 毛玻璃效果

```css
backdrop-blur-apple {
  backdrop-filter: blur(20px) saturate(180%);
}
```

应用于：
- 导航栏
- 悬浮卡片
- 模态框

### 2. 平滑过渡

所有元素默认添加过渡：
```css
transition: color, background-color, border-color, transform, opacity;
duration: 150ms;
timing: cubic-bezier(0.4, 0, 0.2, 1);
```

### 3. 悬停效果

**按钮**
- 颜色变化
- 阴影增强
- 无缩放（保持稳定）

**卡片**
- 轻微缩放 (scale-[1.02])
- 边框高亮
- 图片放大

**链接**
- 颜色过渡
- 无下划线
- 简洁优雅

### 4. 响应式设计

**断点**
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

**最大宽度**
- 内容区: 980px（苹果标准）
- 居中对齐
- 两侧留白
